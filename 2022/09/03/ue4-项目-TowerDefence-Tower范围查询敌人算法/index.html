ue4-项目-TowerDefence-Tower范围查询敌人算法

发表于 2022-09-03 更新于 2022-09-08 分类于 unreal engine

查询敌人算法

思路

  1. 在上一节中的怪物查询敌人算法里用到了，在这一节也需要用到

  2. 所以在TowerDefence\Source\TowerDefence位置创建StoneDefenceUtils.h以及StoneDefenceUtils.cpp

  3. 将MonsterAIController.cpp中的那个lambda函数的逻辑挪动到StoneDefenceUtils.cpp内

    • 函数名称为FindTargetRecently

    • 形参除了原本的数组之外，还需要添加一个const FVector类型的参数Loc用来表示自己的位置

    • 之前形参中的数组，数据类型修改为基类ARuleOfTheCharacter

    • 修改一些变量的名称，使其更加合理

  4. 之后要查询敌人，就直接调用StoneDefenceUtils.cpp中的函数FindTargetRecently即可

思路

  1. 在TowersAIController.h中

    • 声明一个数组，用于存储怪物实例TArray<ARuleOfTheCharacter*>TArrayMonsters;

      • 但是记得需要声明宏UPROPERTY()

      • 为啥元素类型是ARuleOfTheCharacter*呢？因为这是Monster的基类，并且在已经写好的FindTargetRecently()函数中，返回值是ARuleOfTheCharacter*

    • 声明函数virtual AActor* FindTarget();专门用于查询敌人在哪

    • 声明函数void BTService_FindTarget();用于更新黑板中的变量的值

    • 声明构造函数

    • 声明重载函数virtual void Tick(float DeltaTime) override;

    • 声明一个弱指针TWeakObjectPtr<class ARuleOfTheCharacter>Target;

    • 声明一个float类型变量HeartbeatDiagnosis

  2. 在TowersAIController.cpp中

    • 定义函数BTService_FindTarget()

      • 使用.Empty()函数清空TArrayMonsters数组

      • 使用GetPawn<>()获取ATowers的pawn命名为Towers并且判断是否获取成功

      • 再通过IsActive()判断Towers是否是活的

      • 使用TActorIterator遍历此世界中的AMonster

      • 创建一个AMonster类型的指针TheCharacter存储遍历到的AMonster

      • 再通过IsActive()判断TheMonster是否是活的

      • 创建一个FVector类型的变量TDistance存储TheCharacter和TowersAIController的Pawn之间的距离

      • 判断TDistance.Size() <= 1600

      • 将TheCharacter使用Add()函数添加到TArrayMonsters中

      • 当TArrayMonsters.Num() > 0，则设置Towers->bAttack = true;，否则为false

    • 定义函数virtual AActor* FindTarget()

      • 判断TArrayMonsters是否为空

      • 如果不为空，则调用StoneDefenceUtils::FindTargetRecently()函数

      • 如果为空，则返回空指针

    • 在构造函数中初始化HeartbeatDiagnosis为0.0f

    • 定义函数Tick(float DeltaTime)

      • Super继承父类

      • HeartbeatDiagnosis += DeltaTime;

      • 判断HeartbeatDiagnosis >= 0.5f

      • 是则调用BTService_FindTarget();，并且将HeartbeatDiagnosis重置为0.0f

      • 判断TArrayMonsters.Num()

      • 不为空则再判断ATowers* Towers = GetPawn<ATowers>()

      • 创建成功则判断!Target.IsValid() || !Target->IsActive()

        • 条件满足则调用函数FindTarget()给Target赋值为Cast<ARuleOfTheCharacter>(FindTarget())

        • 判断Target.IsValid()（实现塔的角度旋转）

        • 则设置塔的角度为FRotationMatrix::MakeFromX(Target->GetActorLocation() - GetPawn()->GetActorLocation()).Rotator();

        • 判断塔的旋转角度是否为FRotator::ZeroRotator

        • 如果不是则使用Towers->TowersRotator -= GetPawn()->GetActorRotation();实现塔逐帧旋转到怪物位置

          • Towers->TowersRotator在Tower.h中声明的，在蓝图中更新
