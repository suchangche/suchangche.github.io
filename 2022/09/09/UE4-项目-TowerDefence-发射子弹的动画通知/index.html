UE4-项目-TowerDefence-发射子弹的动画通知

发表于 2022-09-09 分类于 unreal engine

动画通知

知识点

FTransform

  • 存储Location信息以及Rotation信息

  • 分别调用函数SetLocation(Loc)以及SetRotation(Rot.Quaternion())给Location以及Rotation赋值

  • 注意：FTransform中的Rotation必须是四元数类型

  • 需要将FRotator类型的数据调用函数Quaternion()转换为四元数之后才可传入

FActorSpawnParameters

传递给SpawnActor函数的可选参数的结构，有如下成员变量

  • Name：要指定为正在生成的Actor的名称的名称。如果未指定值，则生成的参与者的名称将使用[Class]_[Number]形式自动生成。

  • Template：生成新参与者时用作模板的参与者。生成的Actor将使用模板Actor的属性值初始化。如果保留为空，则类默认对象（CDO）将用于初始化生成的参与者。

  • Owner：产生这个Actor的Actor。（可以保留为空）

  • OverrideLevel：生成演员的ULevel，即此对象所在的UObject。如果保留为空，则使用所有者的对象所在的UObject。如果所有者为空，则使用持久级别

  • Instigator:对所产生的Actor造成的伤害负责的APawn。（可以保留为空）/也就是子弹的枪

SpawnActor<>()

SpawnActor<生成Actor的类>(由谁生成Actor的类，位置旋转信息，ActorSpawnParameters信息)

源代码

思路

  1. 创建一个动画通知，调用攻击函数，在塔攻击动画的某一帧添加该通知，动画播放到该位置的时候就会发射子弹了

  2. 在TowerDefence\Source\TowerDefence\Public\Character\Anim\AnimNotify\位置创建AnimNotif类型的CPP文件AnimNotify_SpawnBullet.h

  3. 在StoneDefenceUtils.h中

    • 声明一个函数SpawnBullet()专门用于生成子弹

      • ARuleOfTheBullet* SpawnBullet(UWorld* World, APawn* NewPawn, UClass* InClass, const FVector& Loc, const FRotator& Rot);

  4. 在StoneDefenceUtils.cpp中

    • 定义函数SpawnBullet()

      • 判断关卡，发射子弹的pawn，子弹类是否有效

      • 创建FTransform类型变量Transform，存储生成子弹的位置和旋转信息

      • 将形式参数Loc使用函数SetLocation()传入Transform中

      • 将形式参数Rot调用函数Quaternion()转换为四元数之后使用SetRotation()传递给Transform

      • 创建一个FActorSpawnParameters类型变量ActorSpawnParameters作为后面要用到的SpawnActor<>()的传入参数

      • 使用SpawnActor<>()创建一个ARuleOfTheBullet*类型的变量Bullet

      • 创建成功之后返回Bullet

  5. 在AnimNotify_SpawnBullet.h中

    • 声明构造函数

    • 声明重载函数virtual FString GetNotifyName_Implementation() const override

    • 声明重载函数virtual void Notify(USkeletalMeshComponent* MeshComp, UAnimSequenceBase* Animation) override;

    • 声明TSubclassOf<ARuleOfTheBullet> BulletClass;

    • 声明FName InSocketName;作为骨骼名称

  6. 在AnimNotify_SpawnBullet.cpp中

    • 构造函数中

      • 初始化继承父类

      • #if WITH_EDITORONLY_DATA表明如果实在编辑器中

        • 赋值InSocketName为OpenFire——InSocketName = TEXT("OpenFire");

        • 给动画通知上个色NotifyColor = FColor(196, 142, 255, 255);

        • #endif

    • GetNotifyName_Implementation()函数中

      • 判断BulletClass是否生成成功

      • 是则return BulletClass->GetName()

      • 否则return Super::GetNotifyName_Implementation()

    • Notify(USkeletalMeshComponent* MeshComp, UAnimSequenceBase* Animation)函数中

      • #if WITH_EDITOR表示在动画编辑器中

        • 创建FVector类型变量ComponentLocation赋值为MeshComp的名为InSocketName骨骼的位置信息

        • 创建FRotator类型变量ComponentRotation赋值为MeshComp的名为InSocketName骨骼的旋转信息

      • #else表示在游戏中

        • FVector ComponentLocation = Character->GetFirePoint()->GetComponentLocation();

        • FRotator ComponentRotation = Character->GetFirePoint()->GetComponentRotation();

      • #endif

      • 使用GetOuter()函数获取MeshComp的UObject，赋值给AActor*类型的变量Character

      • 调用StoneDefenceUtils::SpawnBullet()函数，生成ARuleOfTheBullet* Bullet

      • 参数分别为Character->GetWorld()，Cast<APawn>(Character)，BulletClass，ComponentLocation，ComponentRotation

  7. 将塔的攻击动画中的合适位置添加此通知

  8. 调整好塔的骨骼OpenFire的位置和旋转信息
