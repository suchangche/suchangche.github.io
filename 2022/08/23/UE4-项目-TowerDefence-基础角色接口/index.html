UE4-项目-TowerDefence-基础角色接口

发表于 2022-08-23 分类于 unreal engine

基础角色接口

角色基类就是RuleOfTheCharacter，在这里定义了一切角色通用的基础信息（生命值/攻击力/碰撞/模型等）

知识点

FORCEINLINE

强制内联函数

TakeDamage()

虚幻内置的伤害函数，可以自己重载

SetCollisionProfileName()

设置碰撞检测通道

SetBoxExtent()

设置碰撞检测盒子大小

meta = (AllowPrivateAccess = “true”)

让一些组件的细节面板私有化，不和actor本身的堆在一起

大致步骤

  1. 在TowerDefence\Public\Interface\Character位置创建一个IRuleCharacter.h，用于写一些通用的接口

    • 引入虚幻宏与头文件#pragma once，#include "CoreMinimal.h"

    • 在class IRuleCharacter中

      • public:声明以下函数

        • virtual bool IsDeath() = 0;//=0意味着在子类中必须实现，否则报错

        • virtual float GetHeath() = 0;

        • virtual float GetMaxHeath() = 0;

        • virtual bool IsTeam() = 0;

  2. 在RuleOfTheCharacter.h中，Public:IRuleCharacter

    • 宏声明UPROPERTY(VisibleAnyWhere, BlueprintReadOnly, Category = "BaseAttrbute", meta = (AllowPrivateAccess="true"))

      • 声明class USceneComponent* HomingPoint;，实现子弹跟踪效果（跟踪点）

      • 声明class UWidgetComponent* Widget;，实现显示等级，生命值等信息

      • 声明class UArrowComponent* OpenFirePoint;，实现角色及怪物子弹生成位置

      • 声明class UBoxComponent* TraceShowCharacterInformattion;，用于捕捉鼠标射线

    • protected:下声明重载TakeDamage()函数

      • virtual float TakeDamage(float Damage, struct FDamageEvent const& DamageEvent, AController* EventInstigator, AActor* DamageCauser) override;

      • Damage为伤害值

      • DamageEvent为伤害类型

      • EventInstigator为伤害当前Controller的Actor（被伤害目标）

      • DamageCauser为施加伤害的Actor

    • 声明继承自IRuleCharacter的四个函数

      • virtual bool IsDeath();

      • virtual float GetHeath();

      • virtual float GetMaxHeath();

      • virtual bool IsTeam();

    • 在public:下声明以下函数

      • FORCEINLINE ATowerDefencePlayerController* GetGameController()

        • 直接定义

        • return GetWorld() ? GetWorld()->GetFirstPlayerController<ATowerDefencePlayerController>() : NULL;

        • 获取当前world，成功则设置当前世界GetFirstPlayerController为ATowerDefencePlayerController，否则返回空指针

      • FORCEINLINE ATowerDefenceGameState* GetGameState()

        • 直接定义

        • return GetWorld() ? GetWorld()->GetGameState<ATowerDefenceGameState>() : NULL;

        • 获取当前world，成功则设置当前世界GetGameState为ATowerDefenceGameState，否则返回空指针

      • FORCEINLINE USceneComponent* GetHomingPoint() const

        • return HomingPoint;

      • FORCEINLINE UArrowComponent* GetFirePoint() const

        • return OpenFirePoint;

  3. 在RuleOfTheCharacter.cpp中

    • ARuleOfTheCharacter::ARuleOfTheCharacter()中

      • 分别使用CreateDefaultSubobject创建HomingPoint，Widget，OpenFirePoint，TraceShowCharacterInformattion

      • 使用AttachToComponent将上面创建的四个实例绑定到根组件上

      • 使用SetCollisionProfileName()设置TraceShowCharacterInformattion的碰撞通道为Scanning

      • 使用SetBoxExtent设置TraceShowCharacterInformattion的碰撞检测盒子大小为FVector(38, 38, 100)

  4. 在Towers.h中

    • 具体定义

    • 宏声明UPROPERTY(VisibleAnyWhere, BlueprintReadOnly, Category = "BaseAttrubute", meta = (AllowPrivateAccess = "true"))

      • 声明class UParticleSystemComponent* ParticleMesh;

        • 粒子特效组件（在这里就是塔模型）

      • 声明class UStaticMeshComponent* StaticMeshBuilding;

        • 用于显示某处是否可建造（true则显示绿色，否则红色）

      • 声明class UDestructibleComponent* DestructibleMeshBuilding;

        • 破碎效果组件（塔被摧毁之后切换为此）

    • public:声明构造函数

    • protected:声明重载TakeDamage函数

      • virtual float TakeDamage(float DamageAmount, struct FDamageEvent const& DamageEvent, class AController* EventInstigator, AActor* DamageCauser) override;

  5. 在Towers.h中

    • 构造函数中

      • 使用CreateDefaultSubobject

        • 创建ParticleMesh

        • 创建StaticMeshBuilding

        • 创建DestructibleMeshBuilding

      • 使用AttachToComponent将以上三个实例绑定到根组件组件

    • ATowers::TakeDamage()

      • 继承父类

      • 返回值暂定为0

  6. 在Monsters.h中

    • 声明重载函数BeginPlay()以及Tick(float DeltaTime)

  7. 在Monsters.cpp中

    • 重载的函数继承父类

  8. 在TowerDefence.Build.cs中

    • 添加模块"ApexDestruction"

  9. 在TowerDefence.uproject中

    • 添加插件模块ApexDestruction（或者直接在编辑器中启用ApexDestruction插件）
