UE4-项目-TowerDefence-编写ScreenLoading

发表于 2022-08-11 更新于 2022-08-18 分类于 unreal engine

这就是登录游戏的时候的加载界面

为了保证这个Loading可以被反复使用（其他项目也可以用），所以通过写插件的方式实现

Slate简单讲解

Slate是什么？

Slate是UE4的底层UI框架。我们经常使用的UMG和UE4编辑器的各个界面底层都是Slate。

SWidget类

SWidget是所有Slate控件的基类

  • SCompoundWidget可以放一个子控件
  • SLeafWidget不能放子控件
  • SPanel可以放多个子控件

Slate的组成

声明

宏 SLATE_BEGIN_ARGS 和 宏 SLATE_END_ARGS 

在以上两者之间的声明参数，常用的参数类型有 

  • SLATE_ATTRIBUTE(属性)

  • SLATE_EVENT(事件)

  • SLATE_ARGUMENT(参数)

  • SLATE_NAMED_SLOT(插槽) 和 SLATE_DEFAULT_SLOT

    • 声明格式：SLATE_ATTRIBUTE（变量类型，变量名）

Slate语法

Slate的语法比较特殊，不过可读性很强，注意它的特殊缩进格式，如果使用vs的格式化快捷键会破坏掉缩进格式要小心

SNew(WidgetType) 作用是创建一个SWidget。STextBlock、SButton、SVerticalBox的功能和我们经常使用的UMG里的控件差不多。Slot直译为插槽，通过增加插槽我们可以嵌套其他SWidget。

总结

  • HUD类：以前的UI解决方案，现在很少用了。

  • SWidget类图：三大类，SCompoundWidget能放一个子Widget，SLeafWidget不能放子Widget，SPanelWidget能放很多个。

  • Slate特殊写法：SNew新建，加点设置参数，加Slot设置子Widget。

  • TSharePtr和TShareRef智能指针：TSharefPtr可以为空，可以在头文件定义。

创建插件

知识点

FLoadingScreenAttributes

一种数据类型，Loading界面的所有属性值都放在这里面了，声明一个此类型的变量，设置其中具体值，再赋值给SetupLoadingScreen()函数即可

  1. bAutoCompleteWhenLoadingCompletes

    • 如果为真，则在播放所有电影并完成加载后，加载屏幕将立即消失,此时进入游戏

  2. bMoviesAreSkippable

    • 如果为true，则只要加载完成，就可以通过单击加载屏幕跳过电影

  3. bWaitForManualStop

    • 如果为true，播放的动画（幻灯片）会一直循环，直到关卡加载好

  4. PlaybackType

    • 播放类型，表示我们是否应该循环播放等

  5. WidgetLoadingScreen

    • 要显示在电影顶部的小部件，如果没有电影，则只需单独显示

    • 使用SNew传入一个slate并且赋值给WidgetLoadingScreen

EMoviePlaybackType

定义了动画渲染模式，是循环播放，还是仅仅播放一次

  1. MT_Normal

    • 正常播放模式。播放播放列表中的每部电影一次

  2. MT_Looped

    • 循环播放模式。按顺序播放播放列表中的所有电影，然后重新开始，直到手动取消

  3. MT_LoadingLoop

    • 交替循环模式。播放播放列表中的所有电影，并循环播放最后一部电影，直到加载完成

GetMoviePlayer()

获取动画播放器

  1. GetMoviePlayer()->SetupLoadingScreen()

    • 设置加载界面，传入FLoadingScreenAttributes类型的参数

  2. GetMoviePlayer()->PlayMovie()

    • 开始播放动画

大致步骤

  1. 编辑->插件->新插件->空插件->SimpleScreenLoading

  2. TowerDefence\Plugins\SimpleScreenLoading\Source\SimpleScreenLoading\Public路径下创建文件SScreenLoading.h，SScreenLoading.h

  3. TowerDefence\Plugins\SimpleScreenLoading\Source\SimpleScreenLoading\Private路径下创建文件SScreenLoading.cpp，SScreenLoading.CPP

  4. 在SScreenLoading.h中

    • class SScreenLoading继承自SCompoundWidget基类

    • 添加slate的标准三件套：SLATE_BEGIN_ARGS(){}，SLATE_END_ARGS()，void Construct()

      • SLATE_BEGIN_ARGS(SScreenLoading)

      • 构造函数传入参数void Construct(const FArguments& InArgs,const FString &InMapName)

    • 创建一个函数GetProgress()用于获取进度条的进度

      • 返回值类型为TOptional<float>

        • 当我们有一个可选值时， IsSet()返回 true，而GetValue()是有意义的。否则GetValue()没有意义。

        • 当返回值是一个有效的float的时候，则返回值有意义，否则无意义

      • 是一个const函数

    • 定义一个STring类型的变量MapName

  5. 在SScreenLoading.cpp中

    • Construct()中

      • 将参数InMapName赋值给MapName

      • 使用ChildSlot[]创建一个子插件

        • 使用SNew创建一个类型为SProgressBar的组件

        • 使用.percent()进行代理绑定函数GetProgress

          • SNew(SProgressBar).Percent(this,&SScreenLoading::GetProgress)

      • 定义函数GetProgress()

      • 异步获取加载地图的进度百分比，并且使用一个float类型的变量LoadPrecentage保存起来

        • float LoadPrecentage = GetAsyncLoadPercentage(*MapName);

      • 返回LoadPrecentage* 0.01

  6. 在SimpleScreenLoading.h中（调用模块，在模块中实现具体功能）

    • 在private:下声明函数

      • void BeginLoadingScreen(const FString& MapName)

      • void EndLoadingScreen(UWorld* World);

    • 在public:下声明函数

      • SIMPLESCREENLOADING_API void SetupScreenLoading();

        • 前面添加的SIMPLESCREENLOADING_API（文件名_API）是导出宏，生成一个DLL文件，方便在其他位置（工程文件中/非插件文件位置）调用该函数

  7. 在SimpleScreenLoading.cpp中

    • 在BeginLoadingScreen(const FString& MapName)函数中

      • 定义一个FLoadingScreenAttributes类型的参数LoadingScreenArg

      • 设置其bAutoCompleteWhenLoadingCompletes为false

      • 设置其bMoviesAreSkippable为true

      • 设置其bWaitForManualStop为false

      • 设置其PlaybackType为EMoviePlaybackType::MT_LoadingLoop

      • 设置其WidgetLoadingScreen为SNew(SScreenLoading,MapName)

      • 调用函数GetMoviePlayer()，以此函数调用函数SetupLoadingScreen()并且将LoadingScreenArg作为参数传入

      • 调用函数GetMoviePlayer()，以此函数调用函数PlayMovie()

    • SetupScreenLoading()函数中

      • 设置全局CoreUObject委托，PreLoadMap预加载地图使用AddRaw添加代理，绑定BeginLoadingScreen

      • 设置全局CoreUObject委托，PostLoadMapWithWorld使用AddRaw添加代理，绑定EndLoadingScreen

  8. 在SimpleScreenLoading.Build.cs文件中

    • 添加模块 "MoviePlayer"

  9. 在TowerDefenceGameInstance.h文件中

    • 重载函数Init()

  10. 在TowerDefenceGameInstance.cpp文件中

    • 定义函数Init()

      • Super::继承父类

      • 创建一个FSimpleScreenLoadingModule类型的变量SimpleScreenLoadingModule

        • FSimpleScreenLoadingModule类型就是插件模块类型

      • 使用FModuleManager，并且使用LoadModuleChecked加载一个FSimpleScreenLoadingModule类型的模块SimpleScreenLoading（也就是刚刚写好的插件）

      • 通过SimpleScreenLoadingModule调用函数SetupScreenLoading()

  11. 在TowerDefence.Build.cs中

    • 添加模块"SimpleScreenLoading"
