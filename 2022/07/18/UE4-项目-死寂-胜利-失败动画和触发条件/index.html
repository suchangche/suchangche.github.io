UE4-项目-死寂-胜利/失败动画和触发条件

发表于 2022-07-18 分类于 unreal engine

胜利动画和触发条件

知识点

从父项中移除

可以关闭当前UI界面

获取当前关卡名

字面意思

show mouse cursor

显示鼠标

设置输入模式游戏和UI

设置游戏和UI的输入模式，可以使得”Hide Cursor During Capture“引脚变换

思路

  1. 创建控件蓝图“胜利”

    • 添加“边界”，设置其锚点，颜色，透明度

    • 添加图像，设置锚点，尺寸，位移

    • 添加两个文本“成功”，“获得清道夫成就”，分别设置其锚点，位置，尺寸

  2. 添加两个“按钮交互”分别命名为“重新开始”和“返回主界面”

    • 添加动画“出现”

      • 添加边界轨道，设置其渲染不透明度

      • 添加两个文本和图片轨道，设置渲染不透明度和平移

      • 添加按钮轨道，添加渲染不透明度和平移

  3. 在图表中，事件构造之后直接播放动画“出现”

    • 获取“返回主界面”按钮，获取其button，绑定事件到点击时，自定义事件“返回主界面事件”，然后“打开关卡（按名称）”，名称为主菜单关卡名称

    • 获取“重新开始”按钮，获取其button，绑定事件到剪辑师，自定义事件“重新开始事件”，”获取当前关卡名“，创建控件，class选择“加载界面”，将其添加到视口，然后使用“从父项中移除”来关闭当前的胜利UI

  4. 在关卡蓝图中创建自定义事件“胜利”

    • 获取玩家控制器之后“使用混合设置视图目标”，切换一个摄像机，并且“禁用输入”，调用玩家角色蓝图，调用“停止射击”事件

    • 通过“获取类的所有控件”获取“游戏中界面”，“从父项中移除”，并且延迟5s

    • 使用“创建控件”，“添加到视口”将“胜利”控件UI添加到视口

    • 最后 使用获取玩家控制器之后调用”show mouse cursor“为true，并且”设置输入模式游戏和UI“，将Hide Cursor During Capture取消勾选

  5. 在”游戏中界面“的图表中的”当前丧尸数量“函数内，判断当前丧尸数量如果<=0则”Do once“，然后“执行控制台命令”：ce 胜利

失败动画和触发条件

思路

  1. 复制”胜利“控件蓝图，重命名为”失败“

  2. 修改其中的图片和文字

  3. 在角色死亡系统图表中，关闭角色碰撞后面，通过”获取类的所有控件“和”从父项中移除“来关闭”游戏中界面“

  4. 最后在角色死亡弹簧臂拉长的时间轴结束之后（finish引脚）创建”失败“控件添加到视口

  5. 使用获取玩家控制器之后调用”show mouse cursor“为true，并且”设置输入模式游戏和UI“，将Hide Cursor During Capture取消勾选
