添加函数实现视角变大变小

  1. 在TowerDefenceGameCamera.h中声明函数void Zoom(bool bDirection, float& ZoomSpeed);

  2. 在cpp中定义

    • bDirection表示缩放方向，true则为缩小，false则为放大；所以先通过if()判断bDirection的值

    • 为true则再判断弹簧臂的长度，>400则CameraBoom->TargetArmLength -= ZoomSpeed * 2;

    • 为false则判断弹簧臂的长度，<800则CameraBoom->TargetArmLength += ZoomSpeed * 2;

  3. 在TowerDefencePlayerController.h中声明两个函数void MouseWheelUp();以及void MouseWheelDown();

  4. 在cpp中定义

    • 声明一个ATwoerDefenceGameCamera类型的实例zoomCamera，并且通过cate<>()调用其中的成员函数GetPawn()

    • 判断ZoomCamera是否存在，然后调用其中刚写好的函数Zoom()

    • UP时则为true，down时则为false

主大厅与大厅菜单系统

知识点

NativeConstruct()

UMG界面中的初始化函数，类似于其他位置的BeginPlay()函数

UPROPERTY(meta=(BindWidget))

代表以下C++控件和蓝图中的控件绑定

注意：命名需要相同

大致步骤

  1. 创建以RuleOfTheWidget为父类的C++类UUI_HallMenuSystem

  2. 在UI_MainHall.h中

    • 调用宏UPROPERTY(meta = (BindWidget))之后获取UUI_HallMenuSystem的实例HallMenuSystem

    • 在public后声明重载NativeConstruct()函数

    • 在cpp文件中定义NativeConstruct()函数，添加上Super::NativeConstruct()

  3. 在UUI_HallMenuSystem.h中

    • 声明一个类UButton
    • 调用UPROPERTY(meta = (BindWidget))宏，分别声明UButton类型的实例
      • GameStartButton
      • HistoryButton
      • GameSettingsButton
      • TutorialWebsiteButton
      • BrowserButton
      • QuitGameButton
    • 在public下声明重载函数NativeConstruct()
    • 分别声明函数
      • void GameStart();
      • void History();
      • void GameSettings();
      • void TutorialWebsite();
      • void Browser();
      • void QuitGame();

  4. 在UUI_HallMenuSystem.cpp中

    • NativeConstruct()中使用Super::声明继承父类

    • 使用OnClicked.AddDynamic(this, &UUUI_HallMenuSystem::GameStart)绑定多播代理

    • 分别将GameStartButton,HistoryButton,GameSettingsButton,TutorialWebsiteButton,BrowserButton,QuitGameButton

  5. 在Content/UI/Hall/System路径下创建HallMenuSystem_BP控件蓝图

    • 修改父类为UI_MainHall

    • 设置大小为自定义，设置宽度和高度分别为401和501

    • 添加一个边界

    • 边界中添加六个按钮

    • 每个按钮中添加一个文本

    • 修改六个按钮名称，使其对应UUI_HallMenuSystem.h中声明的六个UButton类型实例名称

    • 文本亦然

    • 设置六个按钮之间的填充间距为10

  6. 在HallMain_BP中添加自定义控件HallMenuSystem_BP

    • 设置其锚点和位置在界面右下角

  7. 在HallMap地图中运行即可看见Hall大厅
